
# Pattern Generator 

## Description
This program generates different star patterns using nested loops.

## Patterns Included
- Pyramid
- Inverted Pyramid
- Diamond

## Concepts Used
- Nested loops
- Pattern logic
- Method modularization

## How to Run
1. Compile: javac PatternGenerator.java  
2. Run: java PatternGenerator

## Learning Outcome
Improved logical thinking and loop control in Java.
